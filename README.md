メモ的な中身 
---------

0x0000 - 0x007f : 1byte  
0x0080 - 0x07ff : 2byte  
0x0800 - 0xffff : 3byte  

LF(0x0A)は2byteで計算  
CR(0x0D)はどうも入ってきてないみたい？　->　とりあえず無視ってる
